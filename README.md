# Campaign Analytics Dashboard

Full stack dashboard using React, FastAPI, and SQLite.

## Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: FastAPI
- Database: SQLite
- Deployment: Vercel & Railway

## Features

- Campaign listing
- Status filter
- REST API
- Responsive UI

## Run Locally

### Backend
cd backend  
venv\Scripts\activate  
python -m uvicorn main:app --reload  

### Frontend
cd frontend  
npm install  
npm run dev  

## API

GET /campaigns
